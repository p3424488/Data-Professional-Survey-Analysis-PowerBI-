**📊 Data Professional Survey Analysis – Power BI
📌 Project Overview**

This project analyzes a Data Professional Survey dataset to understand trends in the data industry.
Using Power BI, the dataset was cleaned, transformed, and visualized to uncover insights about salaries, job roles, programming languages, and job satisfaction among data professionals.

The goal of this project is to demonstrate data analysis, data visualization, and business intelligence skills by converting raw survey data into meaningful insights.

**📂 Dataset**

The dataset contains responses from data professionals across different roles and countries. It includes information about:

Job Title

Salary

Country of Residence

Favorite Programming Language

Difficulty in entering the data field

Work-life balance satisfaction

Salary satisfaction

🛠 Tools & Technologies

Power BI – Data visualization and dashboard creation

Excel / CSV – Data source

Power Query – Data cleaning and transformation

DAX – Calculations and measures

**📊 Dashboard Features**

The Power BI dashboard provides insights such as:

🌍 Country Distribution of Data Professionals

💼 Average Salary by Job Role

💻 Most Popular Programming Languages

📈 Difficulty of Breaking into Data Industry

😊 Work-life Balance Satisfaction

💰 Salary Satisfaction

These visualizations help understand industry trends and professional experiences in the data field.

**🔍 Key Insights**

Some major insights from the analysis include:

Data Scientists and Data Engineers generally earn higher average salaries compared to other roles.

Python is the most preferred programming language among data professionals.

Many respondents reported moderate difficulty in entering the data industry.

Work-life balance satisfaction varies across different roles.
